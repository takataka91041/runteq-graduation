# runteq-graduation

## サービス概要
技術記事や学習用URLを登録し、
「あとで読む」で終わらせず学習状況を管理するためのWebアプリです。

学習した記事を記録し、学習ステータスを可視化することで、
インプットの抜け漏れを防ぐことを目的としています。

## 想定ユーザー
- 技術記事を読む機会が多いエンジニア・学習者
- Qiita / Zenn / ブログ記事を溜めがちな人
